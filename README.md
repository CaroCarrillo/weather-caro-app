# Weather App

Proyecto React + Vite preparado para ejecutarse sin instalar Node en tu equipo local.

## Opcion recomendada: GitHub Codespaces

1. Sube esta carpeta a un repositorio en GitHub.
2. En GitHub, abre el repo y elige:
   - Code
   - Codespaces
   - Create codespace on main
3. Espera a que termine la creacion del entorno.
4. En la terminal del Codespace ejecuta:

```bash
npm run dev
```

5. Abre el puerto 5173 cuando GitHub lo sugiera.

## Scripts utiles

- Desarrollo local/remoto:

```bash
npm run dev
```

- Desarrollo forzando host y puerto:

```bash
npm run dev:remote
```

- Build:

```bash
npm run build
```

- Preview:

```bash
npm run preview
```

- Preview remoto:

```bash
npm run preview:remote
```

## Docker (si en algun momento te lo habilitan)

```bash
docker compose up --build
```

Abre http://localhost:5173
